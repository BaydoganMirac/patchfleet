# Gemini entry point

Read [AGENTS.md](AGENTS.md) first. It is the canonical project rule set.

Project skills live under [.agents/skills](.agents/skills). Before changing a
provider adapter, local state, the Cloud boundary, or delegated team work, read
the matching SKILL.md listed in AGENTS.md.

Do not create a second rule set for Gemini. Update the canonical files instead.
